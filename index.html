PORTFOLIO

Idea Collection

A vanilla JS fullstack web app that keeps track of all of my ideas for web apps, mobile apps, and any other creative outlets. I am able to create, read, update, and delete entries — all actions are password protected. The website is fully responsive. Deployed to an AWS EC2.

Point Sails — Sail Quoter

Frontend widget embedded in a Weebly site that allows customers to quote a price for a sailboat sail. Implemented in HTML, CSS, and JS. Backend server developed in Node.js and deployed to Heroku.

Stock Chart

A stock charting application that displays 15 trading-days worth of historical prices. The application was developed using p5.js library - courtesy of the Processing Foundation. Data is pulled from the Alpha Vantage equity API.
